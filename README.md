##### 文件部署，整个文件部署到服务器上，比如本地apache默认是htdocs文件夹下，解压htdocs/Search/全部文件，然后修改demo.html中的服务器地址

### 一，查询医生姓名

##### 前端请求文件 search_doctor_name.php

##### 处理类 class_doctor_name.php

##### 前端请求名：doctorName

##### 后端应答：查询有结果：返回JSON对象,包含医院表中的所有信息 [{...}{...}{...}]

#####                    查询无结果：返回-1.
